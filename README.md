# Animated Ice Cream Parlor

![plsticshield](https://shields.io/badge/animation-framer-green?logo=appveyor&style=plastic) ![plsticshield](https://shields.io/badge/library-react-blue?logo=appveyor&style=plastic)  

## [LIVE](https://mathcodes.github.io/animated-ice-cream-parlor/)

## Description: This is a great app to get started usign some interesting animations Framer has to offer. Notice the use of buttons to toggle the state of animation, and how fluid the translations are. 

## Usage
Get the repo on your local machine (fork, clone, download), and use the following commands:
```
yarn 
```
then 
```
yarn start
```

Sure, you can use npm... but why? Hit me up if you prefer npm over yarn as there may be something I'm missing. Anyways, thanks for checking this repo out. Please follow, star, and hit em up!!!!!
